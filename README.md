# Packet Sniffer in Python

A real-time network packet sniffer built in Python using Scapy.
Captures and displays live TCP/UDP traffic on your network.

## Requirements
pip3 install scapy

## Usage
sudo python3 sniffer.py

## Example Output
[1] [TCP] 192.168.1.5:52341 -> 140.82.114.21:443
[2] [UDP] 192.168.1.5:137 -> 192.168.1.255:137

## Skills learned
- Raw socket programming in Python
- TCP/UDP packet analysis
- Real-time network traffic monitoring
- How Wireshark works under the hood
